URL Validator Add-In for Excel

This page provides the Excel add-in and instructions to install and use it to validate URLs in any column of an Excel worksheet.

Download Add-In (.xlam) Download PowerShell Script (.ps1)

Quick Install & Usage

  1. Download both files: the add-in (`URL_Validator.xlam`) and the PowerShell helper (`URLChecker.ps1`).
  2. Save both files in the same folder on your machine (e.g., C:\Users\You\Documents\URLValidator).
  3. Open Excel and install the add-in:
    • File → Options → Add-Ins → Manage: Excel Add-ins → Go → Browse
    • Select the saved `URL_Validator.xlam` and enable it.
  4. Prepare your workbook: place URLs anywhere starting at row 2 (row 1 is header). The add-in will detect URL-like values in any column.
  5. Run the macro: Press Alt+F8, choose RunURLCheck, then Run. Statuses are written one column to the right of each source URL.
  6. After completion, review the `URL_Log` and `URL_Dashboard` worksheets for summary and failures.

Notes & Requirements

  • PowerShell 5.1+ recommended (PowerShell 7+ preferred for performance).
  • If scripts are blocked, enable execution for the current user in an elevated PowerShell prompt: 
    Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
  • If you edit the add-in before saving, ensure the add-in has a path (save it) so the VBA can locate `URLChecker.ps1`.
